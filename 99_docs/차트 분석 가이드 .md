# 차트 분석 가이드 - 초보자를 위한 투자 대시보드 읽는 법

> **이 문서의 목적**: Grafana 대시보드의 각 패널을 어떻게 해석하고, 투자 판단에 활용할 수 있는지 초보자 눈높이에서 설명합니다.

---

# Part 1: 차트 읽기의 기초

## 1.1 차트가 알려주는 것

차트는 **"가격의 역사책"**입니다. 과거에 사람들이 얼마에 사고팔았는지를 시간 순서대로 보여줍니다.

```
📈 차트가 우상향 = 가격이 올랐다 = 산 사람이 이익
📉 차트가 우하향 = 가격이 내렸다 = 산 사람이 손해
📊 차트가 옆으로 = 가격이 횡보 = 방향을 못 정함
```

## 1.2 선 그래프 읽는 법 (Time Series)

대시보드에서 가장 많이 보는 형태입니다.

```
Y축 (세로) = 가격 또는 지표 값
X축 (가로) = 시간 (날짜)

     가격
      ↑
  5000│      ╱──
      │    ╱
  3000│──╱
      │
      └──────────────→ 시간
        2020   2023
```

**읽는 순서**:
1. **전체 방향** 먼저 확인: 우상향? 우하향? 횡보?
2. **급격한 변화** 찾기: 폭등/폭락 구간은 언제?
3. **현재 위치** 확인: 고점 근처? 저점 근처? 중간?

## 1.3 여러 선이 있을 때

같은 차트에 여러 종목이 있으면 **상대적 비교**가 가능합니다.

```
예: 코스피 vs 코스닥 차트
- 코스피가 위에 있다 = 절대 가격이 높다 (단위가 같을 때)
- 두 선이 벌어진다 = 격차가 커진다
- 두 선이 교차한다 = 역전이 일어났다
```

> 💡 **팁**: 가격 단위가 다른 종목을 비교할 때는 "정규화 대시보드"를 사용하세요. 모두 시작점=100으로 맞춰서 수익률 비교가 쉬워집니다.

# Part 2: 대시보드 패널별 분석 가이드

## 2.1 📊 한국 지수/ETF 패널

### 포함 종목
- **코스피**: 한국 대형주 시장 전체
- **코스닥**: 한국 성장주/중소형주 시장
- **KODEX200**: 코스피200 지수를 따라가는 ETF

### 이 차트로 알 수 있는 것

| 상황 | 해석 | 의미 |
|------|------|------|
| 코스피 > 코스닥 상승 | 대형주 강세 | 안정적 시장, 외국인 매수 |
| 코스닥 > 코스피 상승 | 성장주 강세 | 위험선호, 개인 매수 활발 |
| 둘 다 상승 | 전체 시장 호황 | 경기 좋음, 유동성 풍부 |
| 둘 다 하락 | 전체 시장 침체 | 경기 불안, 현금 선호 |

### 👨‍🏫 선생님의 조언
> "코스피와 코스닥의 격차를 보세요. 격차가 벌어지면 시장의 '성격'이 바뀌고 있다는 신호입니다. 코스닥이 코스피를 앞서기 시작하면 '위험을 감수하더라도 성장주에 베팅하자'는 투자자가 많아진 거예요."

## 2.2 📊 미국 지수/ETF 패널

### 포함 종목
- **S&P500**: 미국 대형주 500개 (세계 경제의 바로미터)
- **나스닥**: 미국 기술주 중심 (성장주의 성지)
- **VIX**: 공포지수 (변동성 지표)

### VIX (공포지수) 읽는 법

```
VIX 10~15: 😊 시장이 아주 평온함 (과열 주의)
VIX 15~20: 😐 정상 범위
VIX 20~30: 😰 불안감 상승 (조정 가능성)
VIX 30 이상: 😱 공포 상태 (급락장)
```

### 미국 시장이 중요한 이유

```
미국 시장 → 한국 시장에 영향

[전날 밤 미국]     [다음날 한국]
나스닥 +2%    →    코스닥 +1~2% (따라 상승)
S&P500 -3%   →    코스피 -2~3% (따라 하락)
```

### 👨‍🏫 선생님의 조언
> "미국 시장은 한국 시장의 '날씨 예보'와 같아요. 특히 나스닥의 움직임은 다음날 코스닥에 직접적인 영향을 줍니다. VIX가 갑자기 30을 넘으면 '폭풍 주의보'라고 생각하세요."

## 2.3 📊 경제지표 패널 (GDP, CPI)

### GDP (국내총생산)
나라 전체가 1년간 만들어낸 상품과 서비스의 총 가치

```
GDP 성장 = 경제가 커짐 = 기업 실적 개선 기대 = 주가 상승 요인
GDP 역성장 = 경기 침체 = 기업 실적 악화 = 주가 하락 요인
```

### CPI (소비자물가지수)
물가가 얼마나 올랐는지 보여주는 지표

```
CPI 상승 = 인플레이션 = 중앙은행이 금리 올릴 수 있음
CPI 안정 = 물가 안정 = 금리 동결 또는 인하 가능
```

### GDP와 CPI 함께 보기

| GDP | CPI | 상황 | 투자 환경 |
|-----|-----|------|-----------|
| ↑ | 안정 | **골디락스** (이상적) | 주식 최고의 환경 |
| ↑ | ↑↑ | 과열 | 금리 인상 우려 |
| ↓ | ↑↑ | **스태그플레이션** (최악) | 모든 자산 어려움 |
| ↓ | ↓ | 디플레이션 | 채권/현금 선호 |

### 👨‍🏫 선생님의 조언
> "경제지표는 분기별로 발표되니 주가처럼 매일 볼 필요는 없어요. 하지만 GDP가 꺾이거나 CPI가 급등하면 시장의 '계절'이 바뀔 수 있으니 주목하세요."

## 2.4 📊 금리 사이클 패널

### 금리란?
돈을 빌릴 때 내는 **'돈의 사용료'**입니다.

### 금리 사이클의 4단계

```
        금리 인상기              금리 인하기
           ↗️                      ↘️
    저점 → 상승 → 고점(정점) → 하락 → 저점

    [완화정책]  [긴축정책]  [긴축유지]  [완화전환]
```

### 금리와 자산의 관계

```
🔺 금리 상승 시:
   채권 가격 ↓ (새 채권이 더 좋으니까)
   성장주 ↓ (미래 수익의 현재가치 감소)
   은행주 ↑ (이자 수익 증가)

🔻 금리 하락 시:
   채권 가격 ↑
   성장주 ↑ (저금리로 성장 기대)
   배당주 상대적 매력 ↑
```

### KODEX 국고채3년이 알려주는 것

```
국고채 ETF 가격 상승 = 금리 하락 중
국고채 ETF 가격 하락 = 금리 상승 중
```

이것은 **채권과 금리의 시소 관계** 때문입니다:
- 금리가 오르면 → 기존 채권(낮은 금리)의 매력이 떨어져 → 채권 가격 하락
- 금리가 내리면 → 기존 채권(높은 금리)의 매력이 올라가 → 채권 가격 상승

### 👨‍🏫 선생님의 조언
> "금리는 모든 자산 가격의 '중력'과 같아요. 금리가 높아지면 모든 자산에 하방 압력이 생기고, 낮아지면 상승 동력이 됩니다. KODEX 국고채3년 차트가 꺾이는 지점이 금리 전환점일 수 있어요."

## 2.5 📊 환율 패널

### 원/달러 환율
1달러를 사려면 몇 원이 필요한가?

```
환율 1,300원 = 1달러에 1,300원 필요
환율 1,400원 = 1달러에 1,400원 필요 (원화 약세)
```

### 환율이 오르면? (원화 약세)

| 영향 받는 곳 | 방향 | 이유 |
|-------------|------|------|
| 수출 기업 (삼성전자) | 유리 ✅ | 달러 수익의 원화 환산액 증가 |
| 수입 기업 | 불리 ❌ | 원자재 비용 증가 |
| 해외 투자자 (외국인) | 한국주식 매력 ↓ | 환차손 우려 |
| 미국 ETF 투자자 | 유리 ✅ | 달러 자산 가치 상승 |

### 환율 + 미국금리 함께 보기

```
미국 금리 ↑ → 달러 강세 → 원/달러 환율 ↑
미국 금리 ↓ → 달러 약세 → 원/달러 환율 ↓
```

### 👨‍🏫 선생님의 조언
> "환율이 급등하면 외국인이 한국 주식을 팔 수 있어요. 반대로 환율이 안정되면 외국인 매수가 들어오기 좋은 환경이 됩니다. 환율 1,400원 이상은 '위험 신호'로 봐도 됩니다."

## 2.6 📊 섹터별 패널 (대형주, IT, 엔터/게임)

### 섹터 로테이션이란?
경기 사이클에 따라 잘 나가는 섹터가 바뀌는 현상

```
경기 회복기: 경기민감주 (현대차, POSCO) 상승
경기 확장기: 성장주 (네이버, 카카오) 상승
경기 둔화기: 방어주 (삼성바이오, 배당주) 상승
경기 침체기: 안전자산 (채권, 금) 선호
```

### IT/플랫폼 (네이버, 카카오)

| 상황 | 해석 |
|------|------|
| 코스닥보다 더 상승 | 플랫폼 기업 성장 기대 높음 |
| 시장 대비 약세 | 광고/커머스 매출 우려 |
| 미국 나스닥과 동조 | 글로벌 기술주 센티먼트 반영 |

### 엔터/게임 (HYBE, SM, 크래프톤)

| 상황 | 해석 |
|------|------|
| 전체 시장과 다르게 상승 | 콘텐츠 히트작 기대 |
| 특정 종목만 급등 | 개별 이벤트 (컴백, 신작 출시) |
| 전체적으로 부진 | 한류 성장 둔화 우려 |

### 👨‍🏫 선생님의 조언
> "섹터마다 잘 나가는 '계절'이 있어요. 금리 인상기에는 성장주가 힘들고, 경기 회복기에는 경기민감주가 좋습니다. 모든 섹터가 동시에 오르는 경우는 드물어요."

## 2.7 📊 배당/인컴 패널

### 배당주란?
회사가 번 돈의 일부를 주주에게 나눠주는 주식

### 배당주가 좋은 시기

```
✅ 금리 하락기: 예금 금리 낮아지면 배당주 매력 상승
✅ 시장 불확실: 변동성 클 때 안정적 현금흐름 선호
✅ 경기 둔화기: 성장보다 안정 추구
```

### 커버드콜 ETF (KODEX 미국S&P500 커버드콜 OTM)

**커버드콜이란?**
주식을 보유하면서 콜옵션을 팔아 추가 수익(프리미엄)을 얻는 전략

```
일반 ETF: 주가 상승분만 수익
커버드콜: 주가 상승분 + 옵션 프리미엄 수익

단, 주가가 급등하면 상승분이 제한됨 (옵션이 행사되므로)
```

| 시장 상황 | 커버드콜 ETF |
|-----------|-------------|
| 횡보장 | ✅ 최적 (프리미엄 수익) |
| 완만한 상승장 | ✅ 양호 |
| 급등장 | ❌ 불리 (상승 제한) |
| 급락장 | ⚠️ 손실 (일부 상쇄) |

### 👨‍🏫 선생님의 조언
> "커버드콜 ETF는 '월급 받는 투자'와 비슷해요. 매달 분배금이 나오지만, 주가가 크게 오를 때는 그 상승분을 다 못 가져갑니다. 안정적인 현금흐름을 원하는 분께 적합해요."

## 2.8 📊 안전자산 패널 (채권, 금)

### 언제 안전자산을 볼까?

```
시장이 불안할 때 → 투자자들이 안전자산으로 이동
"위험 회피" (Risk-Off) 모드
```

### KODEX 국고채3년

| 상황 | 채권 ETF | 이유 |
|------|----------|------|
| 금리 인하 기대 | ↑ 상승 | 기존 채권 가치 상승 |
| 금리 인상 우려 | ↓ 하락 | 기존 채권 가치 하락 |
| 주식 폭락 | ↑ 상승 | 안전자산 선호 |

### KODEX 골드선물(H)

금은 **"최후의 안전자산"**이자 **"인플레이션 헤지"** 수단

| 상황 | 금 가격 | 이유 |
|------|---------|------|
| 인플레이션 | ↑ 상승 | 실물자산 가치 보존 |
| 달러 약세 | ↑ 상승 | 달러 대안 수요 |
| 지정학 위기 | ↑ 상승 | 공포 시 금 선호 |
| 금리 급등 | ↓ 하락 | 이자 안 주는 금 매력 ↓ |

### 주식 vs 채권 vs 금 동시 관찰

```
주식 ↓ + 채권 ↑ + 금 ↑ = 전형적인 위험회피 국면
주식 ↑ + 채권 ↓ + 금 ↓ = 위험선호 국면
주식 ↓ + 채권 ↓ + 금 ↑ = 스태그플레이션 우려
```

### 👨‍🏫 선생님의 조언
> "안전자산은 '보험'과 같아요. 평소에는 성과가 안 좋아 보이지만, 시장이 폭락할 때 포트폴리오를 지켜줍니다. 채권과 금이 동시에 오르면 '뭔가 불안한 일이 생겼구나'라고 생각하세요."

# Part 3: 리스크 구간별 차트 해석

## 3.1 🟢 정상장/상승장에서 볼 것

**관찰 포인트**:
1. 성장주(코스닥, 나스닥)가 가치주보다 얼마나 더 오르나?
2. 배당주/채권/금은 소외되고 있나?
3. VIX가 낮게 유지되고 있나?

**정상적인 상승장 패턴**:
```
나스닥 > S&P500 > 코스닥 > 코스피 > 배당주 > 채권 > 금
(성장주)                                    (안전자산)
```

## 3.2 🔴 급락장에서 볼 것

**관찰 포인트**:
1. 어떤 자산이 가장 많이 떨어졌나?
2. 채권과 금이 방어 역할을 했나?
3. 배당주/커버드콜이 성장주 대비 낙폭이 적었나?

**급락장에서 확인할 것**:
```
예: 코로나 급락 (2020.02~03)
나스닥: -30% (가장 큰 하락)
코스피: -25%
배당주: -20% (상대적 방어)
국고채: +3% (반대로 상승!)
금: +8% (안전자산 역할)
```

## 3.3 🟡 금리 전환기에서 볼 것

**금리 인상 → 인하 전환 시**:
1. 채권 ETF가 먼저 반등하는지 확인
2. 성장주가 가치주를 이기기 시작하는지 확인
3. 커버드콜이 상대적으로 부진해지는지 확인

**금리 인하 → 인상 전환 시**:
1. 채권 ETF가 먼저 하락하는지 확인
2. 가치주/금융주가 성장주를 이기기 시작하는지 확인
3. 배당주의 상대적 매력이 떨어지는지 확인
